# Atiti'z Nails Website

This is the source code for the Atiti'z Nails website, built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

-   `src/components`: Contains all UI components (Navbar, Hero, Services, About, Footer).
-   `src/App.jsx`: Main application component assembling the layout.
-   `src/index.css`: Tailwind directives and global styles.

## Customization

-   **Colors**: The primary (Deep Purple) and secondary (Gold) colors are defined in `tailwind.config.js`.
-   **Content**: Edit the text in the respective component files in `src/components/`.
